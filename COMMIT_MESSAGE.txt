feat: Add CI wheels and enhance Python API with pandas/polars integration

This commit implements comprehensive Python wheel CI/CD and validates
the existing pandas/polars zero-copy Arrow integration.

Changes:
--------

1. CI/CD Pipeline (.github/workflows/ci.yml)
   - Added `python-wheels` job: builds wheels for Linux, macOS, Windows
   - Added `python-sdist` job: builds source distribution
   - Added `python-test` job: tests wheels on 15 platform/Python combos
   - Uses maturin-action@v1 for efficient cross-platform builds
   - Supports Python 3.8-3.12 via stable ABI (abi3)

2. Python API (Already Implemented, Now Verified)
   - Zero-copy Arrow export via `to_arrow()` using C Data Interface
   - Pandas integration via `to_pandas()` (converts through Arrow)
   - Polars integration via `to_polars()` (converts through Arrow)
   - GIL release during I/O for better concurrency

3. Testing & Documentation
   - Added comprehensive DataFrame integration tests
   - Added working example (examples/dataframe_integration.py)
   - Updated README with installation and usage examples
   - Created IMPLEMENTATION_SUMMARY.md with full details
